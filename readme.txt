== Description ==

Based on _Comments with Social Login__ by Ixtendo.

== Installation ==

This plugin can be installed in one of two ways:

Browsing the WordPress plugin repository
   1. In WordPress Admin Console, click on Plugins >> Add New.
   2. In the search field (located at the right-hand side of the page), search for "Comments with Social Login".
   3. Click the Install Now button.
Uploading the zip archive through WordPress Admin Console
   1. Download the plugin archive from http://www.ixtendo.com/comments-with-social-login-wp-plugin/.
   2. In WordPress Admin Console, click on Plugins >> Add New.
   3. Click on the Upload Plugin button located at the top of the page.
   4. Upload the zip archive and click the Install Now button.
